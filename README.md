=================
# CRYPT

Stupid string encrypter

## Installation

Just clone this repo

    git clone https://github.com/frownfish/CRYPT.git "CRYPT"

## Usage

    python crypt.py (-e|-d) [string]
    
  * -e encrypts
  * -d decrypts
  * If you don't enter a string, the script will prompt you for one.
